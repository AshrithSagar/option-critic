# option-critic

The Option-Critic Architecture

## Installation

Clone the repo and install the [dependencies](requirements.txt):

```shell
pip3 install -r requirements.txt
cd oca
```

## Usage

```shell
python3 main.py
```

```shell
python3 main.py --switch-goal True --env fourrooms
```
